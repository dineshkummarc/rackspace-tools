Tools to control Rackspace cloud servers.

Edit config.json and set your username and api key. Don't remove the auth host as it wont work (because i use node-cloudservers and it's broken).

Do a `npm install .` before using them to install necessary node modules.

Tools:

    list-servers.js - lists cloud servers
